## TO START - Cmder
1. Start mongodb
2. New tab, Connect to mongo, ctrl + c
3. Run database in Postman: cd into main directory, `npm run mon` (this uses nodemon from /bin/www to autoupdate). Open postman.
4. New tab, Start API: cd into client, 2-`npm start`
